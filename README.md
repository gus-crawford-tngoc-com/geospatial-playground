# Geo-space Applications

This app illustrates some very fundamental applications of GeoSpatial searching and indexing in MongoDB.

```
npm start
```

> _The above command will:_ <ul><li>install workspace dependencies</li><li>install Angular dependencies</li><li>start the mongo database and API access in docker</li><li>run the Angular application ([browse to localhost:4201](http://localhost:4201))</li></ul>
## Backend

`docker compose up -d`
